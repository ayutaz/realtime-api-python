# realtime-api-python

OpenAIのreal time apiをpythonで動かすサンプルコード

# requirements
* Python 3.11
* uv

# setup
```
uv sync
```

create .env file
```
OPENAI_API_KEY=your-api-key
```

# run
```
uv run hello.py
```
